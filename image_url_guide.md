# Product Image URL Update Guide

## Instructions
For each product below, find a working image URL from Amazon or Flipkart and update the CSV file.

### How to Get Working Image URLs:
1. Go to **Amazon.in** or **Flipkart.com**
2. Search for the product name
3. Click on the product
4. Right-click the main product image
5. Select "Copy image address" (Chrome/Firefox)
6. Paste the URL into the CSV file

## Steps to Update

### Step 1: Open the CSV File
```
File: supplement_info.csv
Column to Update: supplement image
```

### Step 2: For Each Product
1. Search on Amazon/Flipkart
2. Find the product listing
3. Copy the image URL
4. Replace the old URL in the CSV

### Step 3: Save & Test
- Save the CSV file
- Run your app and check the Supplements page
- Broken images should now display correctly

---

## Alternative: Quick URL Replacement

If you want, copy these trusted sources:
- **Amazon images**: Usually start with `m.media-amazon.com` or `images-na.ssl-images-amazon.com`
- **Flipkart images**: Usually start with `rukminim1.flixcart.com` or `rukminim2.flixcart.com`
- **Direct product sites**: Check manufacturer websites

## Tips
✓ Direct URLs (not redirects) work best  
✓ Avoid Google Shopping encrypted URLs  
✓ Test a few updates to ensure they work  
✓ You can do this incrementally - start with 5-10 products
